AI Research & Analysis Assistant

An AI-powered research assistant that allows users to perform descriptive statistics, inferential statistics, and visualizations on any dataset using natural language commands. It integrates with a web app and WhatsApp, delivering actionable insights and downloadable reports.

Features
1. Natural Language Analysis

-Users can ask questions like:

- Compare business ownership with business sector using chi square

- Plot a countplot of How long has your business been operating?

- The system automatically interprets the columns from user queries using semantic and lexical matching.

2. Statistical Tools

- Descriptive Statistics

- Frequencies, percentages, summary statistics

- Optional LLM-based interpretation in plain English

3. Visualization Tools

- Countplot

Saves plots as PNG for sharing or download

4. Flexible Dataset Handling

- Accepts CSV, Excel, JSON, or list-of-dictionaries

- Column auto-detection (numeric vs categorical)

- Semantic mapping for dynamic datasets (no hardcoding)

5. Downloadable Outputs

- Analysis results can be exported as Excel files with all metrics:

- Frequencies, percentages, Cronbach alpha, p-values, etc.

- Visualizations saved as PNGs

- Suitable for WhatsApp delivery or Streamlit web frontend
